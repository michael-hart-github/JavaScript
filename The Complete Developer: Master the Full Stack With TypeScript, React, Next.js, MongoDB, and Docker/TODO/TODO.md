# Addtional resources to look into:

1. The Coding Workbook by Sam Taylor (No Starch Press, 2020)
   1. teaches you how to build a website with HTML and CSS
2. The Book of CSS3, 2nd edition, by Peter Gasston (No Starch Press, 2014)
   1. sharpen your CSS skills (whatever that is supposed to mean)
3. JavaScript Crash Course by Nick Morgan (No Starch Press, 2024)
   1. familiarize yourself with JavaScript; it is a fast-paced JavaScript tutorial for beginners
4. Eloquent JavaScript, 3rd edition, by Marijn Haverbeke (No Starch Press, 2018)
   1. deep dive into JavaScript
